# 📈 KIS 프리미엄 AI 투자 어시스턴트

한국투자증권(KIS) OpenAPI와 GPT-4o의 강력한 분석 능력을 결합한 **상위 1% VIP 전용 AI 투자 파트너**입니다. 실시간 시장 데이터 조회부터 복잡한 수익률 분석, 국내외 주식 주문까지 대화만으로 간편하게 처리하세요! 😊

---

## 🚀 주요 기능 (Key Features)

### 1. 스마트 시세 및 시장 지수 조회
- **국내/해외 실시간 시세**: 종목명만 말해도 6자리 코드를 자동 검색하여 현재가, 등락률, 주요 재무지표(PER, PBR, EPS)를 즉시 제공합니다.
- **시장 지수 분석**: 코스피(KOSPI), 코스닥(KOSDAQ)의 실시간 지수와 9:00 시가 대비 변동성을 분석합니다.

### 2. 고도화된 자산 및 수익률 분석
- **통합 포트폴리오**: 국내 및 해외 계좌의 총 자산, 보유 종목, 종목별 비중 및 수익률을 한눈에 확인하세요.
- **심층 수익률 진단**: 
  - 특정 기간 동안의 총 자산 변동 및 자산 증감액 분석
  - 일별 수익률 추이 그래프(텍스트형) 제공
  - 실제 매매 내역(실현손익) 교차 검증을 통한 정확한 데이터 제공

### 3. 강력한 주문 관리 (Trading)
- **실시간 즉시 주문**: 원하는 수량과 가격으로 국내외 주식을 대화형으로 매수/매도할 수 있습니다.
- **스마트 예약 주문**: 바쁜 투자자를 위해 미래 특정 날짜에 실행될 예약 주문을 등록하고 관리(조회/취소)합니다.
- **주문 취소**: 미체결 주문이나 예약된 주문을 번거로운 메뉴 클릭 없이 대화로 즉시 취소합니다.

### 4. 시장 통찰력 (Market Intelligence)
- **투자자/공매도 추이**: 외인, 기관, 개인의 매매 동향과 공매도 비중을 분석하여 수급의 흐름을 읽어줍니다.
- **시장 순위 등락**: 등락률 상위, 거래량 상위, 시가총액 상위 종목 리스트를 실시간으로 가져옵니다.
- **실시간 뉴스 검색**: 종목과 관련된 최신 뉴스를 웹에서 검색하여 주가 변동의 원인을 함께 파악합니다.

- **Frontend/Backend**: Next.js 15 (App Router), TypeScript

## ⚙️ 시작하기 (Quick Start)

본 프로젝트를 로컬 환경에서 실행하기 위한 단계별 가이드입니다.

### 1. 환경 변수 설정
프로젝트 루트 디렉토리에 `.env.local` 파일을 생성하고, OpenAI API 사용을 위한 키를 입력합니다.
```env
# .env.local
OPENAI_API_KEY=sk-your-openai-api-key-here
```

### 2. KIS API 설정 파일 (kis_devlp.yaml)
한국투자증권 API 연동을 위한 설정 파일이 필요합니다.
- **경로**: `c:/Users/01999/Documents/source/OpenAPI_trading/kis_devlp.yaml` (또는 프로젝트 상위 디렉토리의 `OpenAPI_trading` 폴더)
- **내용 예시**:
  ```yaml
  my_app: "내 앱키"
  my_sec: "내 앱시크릿"
  my_acct_stock: "계좌번호(8자리)"
  my_prod: "01"
  ```

### 3. 프로젝트 실행 단계
아래 명령어들을 순서대로 터미널에 입력하여 앱을 구동합니다.

```bash
# 1. 의존성 패키지 설치
npm install

# 2. 개발 서버 실행
npm run dev
```

### 4. 접속 및 사용
실행 후 브라우저에서 **[http://localhost:3000](http://localhost:3000)**에 접속하면 스타일리시한 AI 투자 어시스턴트 화면이 나타납니다. 하단 입력창에 "삼성전자 현재가 알려줘" 또는 "오늘 내 수익률 어때?"라고 질문해 보세요! 🚀
